This directory contains all the source code to create the neural network using attention.

There are 3 steps to get the model running:
1. Install the dependencies
2. Train the model
3. Test the model

1. Install the dependencies

To install all the appropriate dependencies, run:
install.sh

That is present in the directory. This assumes you have pip3 and python3 installed.



2. Train the model

Run:
python3 main.py
